Version=---210608 03:25--

An USER is some person that have authority to perform some set of their business tasks into SYSTEM. An USER interacts with SYSTEM through an AGENTs and through only the AGENT. The first task that USER can perform into SYSTEM is proving his/her authority, that are the login and the password. His/her last task is announcement of end of his/her work, that is logout.
This last task causes destroying of the AGENT. That is an AGENT can serve only one USER. Into the SYSTEM, a USER is identified by not negative integer, for example 0, 1, 2 ...
